# imocker

> 初代 2021.04

## 介绍

imocker是一套提供mock数据的本地工具。通过图形化操作界面，为页面提供相应API，提高开发速度



## 主要特色

- 提供可视化界面进行 api 的相关操作
- 服务本地化，随意修改mock数据，不影响其他人操作
- 支持导入 json 串并自动解析参数字段 和 自动生成mock 公式（公式可一键复制）
- 支持 mock 公式直接编辑
- 支持 mock 数据预览



## 使用

启动服务

```bash
npm run serve
```

启动可视化界面

```bash
npm run web
```

浏览器访问地址

```bash
http://127.0.0.1:8001/home
```
